Status Vocabulary

This vocabulary is a re-publication of the Registry Ontology's Status vocabulary (online in RDF). This re-publication has been performed to allow for the URIs of each vocab term (skos:Concept) to resolve to both human-readable and machine-readable forms (HTML and RDF, respectively) using HTTP content negotiation.

Note that just like the original form of this vocabulary, while it is a SKOS vocabulary implemented as a single Concept Scheme, it is also an OWL Ontology and that each Status is both a skos:Concept and a reg:Status.
